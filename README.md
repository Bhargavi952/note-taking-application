# Note-taking-application
## Run Locally

Clone the project

```bash
  https://github.com/Bhargavi952/note-taking-application.git
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm  start
```

## Tech Stack

- React
- Redux 
- Bootstrap
